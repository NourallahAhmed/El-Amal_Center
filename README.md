# El-Amal_Center

It's Flutter application where the center can add patients and schedule sessions 

(( the project still in progress ))


# Features
  - Login
  - Home page
  - Adding Patient

# Tootls
  - MVVM Desgin Pattern
  - Provider
  - FireBase Auth
  - FireBase RealTime DataBase
  - Table Calender Package
  - Weekly Selector Package
